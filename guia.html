📘 Guia Completo

Aqui está tudo o que você precisa saber para usar e criar seus projetos:

  1. Como usar: Digite o que precisa ou envie uma imagem da tela
  2. Criar imagens: Ative o modo "Criar Imagem" e descreva o que quer
  3. Publicar no ar: Salve os arquivos no GitHub → Conecte no Render → Deixe o ponto na pasta
  4. Códigos prontos: Peça e ele gera tudo para você copiar
  5. Gerar PDF: Abra esta página → Imprimir → Salvar como PDF
